Dependencies (Thymeleaf 2.1)

Thymeleaf 2.1 requires Java SE 5.0 or newer. Besides, it depends on the following libraries:

  • unbescape 1.1.0 or later
  • ognl 3.0.8 or later
  • javassist 3.16.1-GA or later
  • slf4j 1.6.6 or later

Additionally, if you use the LEGACYHTML5 template mode, you will need:

  • nekoHTML 1.9.21 or later

Dependencies (Thymeleaf 3.0)

Thymeleaf 3.0 requires Java SE 6.0 or newer. Besides, it depends on the following libraries:

  • attoparser 2.0.0.BETA04 or later
  • unbescape 1.1.1 or later
  • ognl 3.1 or later (only in non-Spring environments)
  • slf4j 1.6.6 or later
