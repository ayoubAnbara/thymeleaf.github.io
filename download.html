Thymeleaf  

Java HTML/XML template engine

The current stable version of Thymeleaf is 2.1.4.RELEASE, released on 22 December 2014.

Maven/Gradle

The easiest way to include Thymeleaf in your project is to use a build system like Maven or Gradle to make use of the Thymeleaf artifacts on Maven. Then you can add the following dependencies to your project and you're away!

  • GroupId: org.thymeleaf
  • ArtifactId: thymeleaf
  • Version: 2.1.4.RELEASE

Include just 1 of the following for either a Spring 3 or 4 project:

  • GroupId: org.thymeleaf
  • ArtifactId: thymeleaf-spring3
  • Version: 2.1.4.RELEASE
  • GroupId: org.thymeleaf
  • ArtifactId: thymeleaf-spring4
  • Version: 2.1.4.RELEASE

And if you want to use the Thymeleaf Testing framework (version does not necessarily match that of core):

  • GroupId: org.thymeleaf
  • ArtifactId: thymeleaf-testing
  • Version: 2.1.3.RELEASE

JARs

JARs for each of the artifacts above can also be downloaded separately from each project's GitHub repository releases page.

  • Thymeleaf: https://github.com/thymeleaf/thymeleaf/releases
  • Thymeleaf Spring: https://github.com/thymeleaf/thymeleaf-spring/releases
  • Thymeleaf Testing: https://github.com/thymeleaf/thymeleaf-testing/releases

When doing this, also be sure to download Thymeleaf's dependencies:

  • Unbescape 1.1.0+
  • OGNL 3.0.8+
  • Javassist 3.16.1-GA+
  • SLF4J 1.6.6+

Additionally, if you're using the LEGACYHTML5 template mode:

  • NekoHTML 1.9.21+
