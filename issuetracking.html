Issue Tracking

We use GitHub for our issue tracking and have several repositories for various parts of the Thymeleaf project, so be sure to use the right one to make any bug reports, raise any issues, or even make pull requests!

Core packages

  • Thymeleaf: https://github.com/thymeleaf/thymeleaf/issues
  • Thymeleaf Spring integration: https://github.com/thymeleaf/thymeleaf-spring/issues
  • Thymeleaf Testing framework: https://github.com/thymeleaf/thymeleaf-testing/issues

Example applications

  • Good Thymes Virtual Grocery: https://github.com/thymeleaf/thymeleafexamples-gtvg/issues
  • Spring Thyme Seedstarter Manager: https://github.com/thymeleaf/thymeleafexamples-stsm/issues
  • ExtraThyme: https://github.com/thymeleaf/thymeleafexamples-extrathyme/issues
  • Thymeleaf vs JSP: https://github.com/thymeleaf/thymeleafexamples-thvsjsp/issues
  • Say Hello!: https://github.com/thymeleaf/thymeleafexamples-sayhello/issues
  • SpringMail: https://github.com/thymeleaf/thymeleafexamples-springmail/issues
  • SpringSecurity: https://github.com/thymeleaf/thymeleafexamples-springsecurity/issues
  • Spring PetClinic + Thymeleaf: https://github.com/thymeleaf/thymeleafexamples-petclinic/issues

Supporting projects
